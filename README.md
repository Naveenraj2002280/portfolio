## 🚀 Featured Production Systems

### 1. AI/BI Querying Engine & Predictive Forecasting Model (Active Platform)
*   **Stack:** Databricks Genie, Databricks AI/BI Dashboards, MLflow, PySpark, Delta Lake
*   **Overview:** Engineered an end-to-end data intelligence solution to resolve complex bottlenecks in querying unstructured retail data. Provisioned a Databricks Genie Space converting natural language queries into valid SQL expressions across massive schemas. Developed a sales forecasting model tracked, versioned, and registered via MLflow workflows.

### 2. Platform Orchestrator – Multi-Cloud Chatbot Automation
*   **Stack:** FastAPI, Terraform, AWS Lambda, SQS FIFO, Azure Functions, GCP Functions, Databricks Genie
*   **Overview:** Built an enterprise-grade automation engine reducing multi-cloud chatbot infrastructure deployments from days to 5 minutes. Engineered an async dual-trigger loop using AWS Lambda and SQS FIFO queues to handle scale webhooks without timeouts, and applied a strict Platform Adapter Pattern separating core logic from API variances.

### 3. Cross-Platform Child Psychology E-Learning App
*   **Stack:** Flutter, Dart, AI Recommendation Engine, Google Ads, E-Commerce
*   **Overview:** Architected a high-performance Android/iOS Flutter app featuring optimized streaming players and integrated e-commerce. Designed and embedded a secure client-side AI recommendation module to deliver personalized clinical course suggestions based on user profiles.

### 4. AI-Powered Service Management Platform
*   **Stack:** Flask API, AWS Bedrock, Google Maps API, Multi-Tenancy Connection Pooling, AWS SES
*   **Overview:** Developed a massive multi-tenant microservices ecosystem consolidating 13+ operational modules. Features an AI-driven scheduling allocator combining AWS Bedrock project estimations with Google Maps transit routing matrices, alongside a mathematical dynamic pricing optimization pipeline linked directly to fleet utilization data.
